CheatSheet Atom

  • ctrl + S : Save
  • ctrl + Q : Quit
  • ctrl + P : Search dans l'arbo
  • ctrl + G : Numéro de ligne
  • ctrl + L : Sélectionner ligne courante
  • ctrl + shift + D : Duppliquer la ligne
  • ctrl + shift + K : Supprimer la ligne
  • ctrl + Z : Annuler la derniere action
  • alt + B : Revenir au début du mot
  • ctrl + shift + H : Github-Plus
  • ctrl + :(/) : Commenter la ligne
